# Putting HOODAL on GitHub

This guide will walk you through the final steps to get your **HOODAL** repository onto GitHub.

### Step 1: Initializing Local Repository (Already Done)
I have already initialized your local git repository and made the initial commit for you.

### Step 2: Create a New Repository on GitHub
1. Go to [github.com/new](https://github.com/new).
2. For **Repository name**, use `HOODAL`.
3. Click **Create repository**.

### Step 3: Connect Local Repository to GitHub
In your terminal, run:
```bash
git remote add origin https://github.com/<your-username>/HOODAL.git
git branch -M main
git push -u origin main
```
Replace `<your-username>` with your GitHub username.
